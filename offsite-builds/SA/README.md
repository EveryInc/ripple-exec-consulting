**Complexity:** Easy | **Estimated time:** 15-20 min to produce useful output

**Connector opportunity:** If SA's calendar is connected, Claude can pull his actual next meeting and prep for it live. That's the "aha" moment — zero effort, immediate value.

### Optional Deep-Dives (Stretch / Follow-Up)

#### 1. Team Update Aggregator & 1:1 Agenda Builder
Ingests team update emails and Slack messages, extracts key themes, blockers, and wins, then generates structured 1:1 meeting agendas tailored to each direct report. Tracks items across weeks so nothing falls through the cracks.

#### 2. Shareholder Info Request Manager
Manages the quarter-end shareholder information request process: tracks what data is needed, who's responsible, deadlines, and completion status. Generates outbound request emails and follow-up reminders automatically.

#### 3. Shareholder Email List Manager
A tool to manage the shareholder contact database: add/remove shareholders, handle multi-email shareholders, validate email addresses, track changes over time, and export clean distribution lists on demand.
