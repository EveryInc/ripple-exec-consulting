# Competitive Intelligence Dashboard

## Prompt

I'm SVP of Strategic Initiatives at Ripple. I need to build a competitive intelligence dashboard that I can update and reference weekly.

Using the attached `competitors.csv` and `competitor_moves.csv` data files, build me an interactive HTML dashboard that includes:

1. **Competitor Overview Matrix** - A comparison table showing each competitor (SWIFT, Stellar, JPMorgan Kinexys, Circle, Visa B2B Connect) vs. Ripple across key dimensions: settlement speed, transaction cost, number of institutional partners, regulatory licenses, geographic coverage, and product breadth.

2. **Recent Competitive Moves Timeline** - A visual timeline of significant competitor announcements, partnerships, product launches, and regulatory developments from the past 90 days. Color-code by threat level (high/medium/low).

3. **Ripple Advantage Scorecard** - For each competitor, show where Ripple leads, where we're at parity, and where we trail. Include a simple red/yellow/green indicator.

4. **Weekly Briefing Generator** - A section that takes the raw data and generates a 3-paragraph executive summary of the most important competitive developments this week, what they mean for Ripple, and recommended actions.

Make the dashboard clean and professional with Ripple's brand colors (dark navy #1a237e, blue #2962ff, white). Include filtering by competitor and by date range. The whole thing should work as a single HTML file I can open in my browser.
